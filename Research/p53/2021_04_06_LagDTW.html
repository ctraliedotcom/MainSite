p21 vs p53 Dynamic Time Warping After DAPS Detrending¶

4/6/2021¶

The code below shows the results of doing constrained dynamic time warping between p53 and p21 with and without DAPS detrending as a pre-processing step. DTW is constrained to stay within a discrepancy of 30 in the warping in either direction.

What we notice is that before detrending, DTW doesn't do much; the identity warp is about as good as any other. But after detrending, we see that we get good warps when p21 lags either 5 behind, or goes about 6 ahead. If we keep in mind that the period of p53 is 11 during oscillation, then this makes perfect sense, since 5 + 6 = 11.

Code to accomplish all of this is below.

def get_dtwdist(filestr, mode='full', Win=0, do_plot=False):
    """
    Compute DTW between p53 and p21 for a particular Gray level
    
    Parameters
    ----------
    filestr: string
        Path to gray time series
    mode: string
        What mode to do cross-correlation in (full/valid/same)
    Win: int
        Window to use for detrending
    do_plot: boolean
        Whether to save a plot for each individual cross-correlation to disk
    
    Returns
    -------
    all_warps: ndarray(N1)
        An array of all shifts from p51 to p21
    peaks: ndarray(N)
        An array of lags, restricted from the range between 0 and 30, that
        led to peaks in each cross-correlation
    """
    p53 = np.loadtxt(filestr.format("p53"))
    p21 = np.loadtxt(filestr.format("p21"))
    divs = np.loadtxt(filestr.format("div"))
    divs = np.sum(divs, 1)
    allndivs = []
    # Histogram of matchings from p53 indices to p21 indices
    I = np.array([])
    J = np.array([])
    N = 0
    all_corr = np.array([])
    if do_plot:
        plt.figure(figsize=(18, 8))
    all_warps = np.array([])
    for i in range(p53.shape[0]): # Loop over all time series
        x1 = p53[i, :]
        x2 = p21[i, :]
        if Win > 0:
            x1 = detrend_timeseries(x1, Win)
            x2 = detrend_timeseries(x2, Win)
        if np.sum(x1 == -1) > 0 or np.sum(x2 == -1) > 0:
            # This signal has missing data, so skip it
            continue
        N += 1
        path = linmdtw.cdtw(x1[:, None], x2[:, None], 30)
        # Record warping path in histogram
        I = np.concatenate((I, path[:, 0]))
        J = np.concatenate((J, path[:, 1]))
        # Find the maximum warp deviation
        all_warps = np.concatenate((all_warps, path[:, 1] - path[:, 0]))
        
        # Plot this individual warping path if requested
        if do_plot:
            plt.clf()
            plt.subplot(231)
            plt.plot(x1)
            plt.title("p53")
            plt.subplot(234)
            plt.plot(x2)
            plt.title("p21")
            plt.subplot(232)
            plt.plot(path[:, 0], path[:, 1])
            plt.xlabel("p53")
            plt.ylabel("p21")
            plt.subplot(235)
            plt.plot(x1[path[:, 0]])
            plt.plot(x2[path[:, 1]])
            plt.legend(["p53", "p21"])
            plt.title("Aligned")
            plt.subplot(233)
            plt.hist(path[:, 0] - path[:, 1])
            plt.title("p53-p21 Lags Histogram")
            plt.savefig("{}.svg".format(i), bbox_inches='tight')
        allndivs.append(divs[i])
    V = np.ones(I.size)
    N = p53.shape[1]
    D = sparse.coo_matrix((V, (I, J)), shape=(N, N))
    return all_warps, np.array(allndivs), D.toarray()
